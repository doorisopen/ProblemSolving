# 스택의 성질
* __FILO(First In Last Out)__
* 출력에 제한이 있어 Restricted Structure 라고 부르기도 한다.
* 원소의 추가가 O(1)
* 원소의 제거가 O(1)
* 제일 상단의 원소 확인이 O(1)
* 제일 상단이 아닌 나머지 원소들의 확인/변경이 원칙적으로 불가능


## 배열을 이용한 스택 구현

```
#include <iostream>
using namespace std;
#define ll long long

const int mx = 1000005;
int dat[mx];
int pos = 0;

void push(int x) { dat[pos++] = x; }
void pop() { pos--; }
int top() { return dat[pos-1]; }

int main() {
    cin.tie(nullptr);
    ios::sync_with_stdio(false);
    push(1);
    push(2);
    push(3);

    cout << top(); // 3
    pop();
    cout << top(); // 2
    
    return 0;
}
```
